# Operating costs

Public record of recurring infrastructure costs for IATF.

This is **not** an audited financial statement. It is an operating cost
log for the public infrastructure used to keep IATF online, kept so that
anyone can see what the organization spends to stay public.

The summary on <https://iatf.cc/transparency/> is generated from these
records.

## What goes here

- Domain registrations and renewals.
- Hosting and infrastructure costs above free-tier.
- Any other recurring public-infrastructure expense.

## What does **not** go here

- Personal addresses, billing details, or unnecessary account
  information. Receipts may be referenced or published after redaction.
- One-off expenses that do not relate to public infrastructure.

## File naming

One file per year: `YYYY.md`.

## Minimal entry shape

```markdown
---
year: 2026
---

| Item   | Cost (USD) | Notes |
| ------ | ---------- | ----- |
| Domain | 3.40       | iatf.cc, funded by mugiwarafx. |
| Hosting | 0          | Free-tier on Cloudflare.       |
```
