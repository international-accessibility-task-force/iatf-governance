# Security policy

This repository holds the public governance record for IATF. It does not
run code in production.

## Reporting a vulnerability

If you find a record here that exposes private information that should
have been redacted (personal contact details, private addresses,
identifying information about individual users of assistive
technologies), please **do not** open a public issue.

Email <contact@iatf.cc> with:

- A link to the entry in question.
- What information looks like it should be removed and why.

We will review and, where appropriate, redact or take down the record
from IATF-controlled surfaces. We cannot guarantee removal from
third-party caches, forks, or screenshots.

For all other security topics, see the org-wide
[SECURITY.md](https://github.com/international-accessibility-task-force/.github/blob/main/SECURITY.md).
